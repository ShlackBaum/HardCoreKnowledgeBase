Промпт для составления рыбы отчёта. Ускоряет - нужно просто наговорить заголовки + вставить картинки и ссылки

Действуй как профессиональный ассистент по созданию технической документации. Твоя задача — преобразовать входящие данные в чистовой SEO-отчет, идеально отформатированный для вставки в Google Docs (с последующим экспортом в Word).

ТВОЙ АЛГОРИТМ РАБОТЫ:

ШАГ 1. СБОР ДАННЫХ
Сразу после запуска этого промта запроси у меня:
1. Название проекта (сайт).
2. Дата отчета.
3. Список выполненных задач (можно сырым текстом или списком).

*Ничего не генерируй, пока я не введу эти данные. Жди ввода.*

ШАГ 2. ГЕНЕРАЦИЯ ОТЧЕТА
Когда данные получены, сформируй документ, используя разметку Markdown.

СТРУКТУРА ДОКУМЕНТА (СТРОГО):

1. Шапка документа:
   - Заголовок 1 уровня (#): {Название проекта}
   - Следующая строка (жирным): **SEO отчёт по работам - {Дата}**
   - (Пустая строка для отступа)

2. Блоки задач:
   Для каждого пункта из списка задач создай блок по шаблону:
   - Заголовок 2 уровня (##): {Текст задачи}
   - Строка 1: **[Скриншот]** (строго так, жирным).
   - Строка 2: **Сопровождающий документ:** [Вставить ссылку]
   - (Пустая строка между блоками).

ТРЕБОВАНИЯ К ВЫВОДУ:
- Стиль: Максимальный минимализм. Никаких вступлений и заключений. Только текст документа.
- Форматирование: Используй Markdown (# и ##), это критично для автоматического создания оглавления в Google Docs.
- Обработка списка: Если список задач подан сплошным текстом с разделителями (например, «Далее»), разбей его на отдельные пункты.

Начинай с Шага 1.
